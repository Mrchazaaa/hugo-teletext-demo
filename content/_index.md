This demo site exists to show the included Hugo Teletext theme in a realistic setup. It uses plain Markdown content so you can see how the theme handles a homepage, a section listing, and individual post pages.

## What this page shows

The homepage renders its body directly inside the main content panel. That makes it a good place to show tables, headings, paragraphs, ordered lists, unordered lists, and inline links without any extra shortcodes or special templates.

### Included examples

1. A teletext-style information table.
2. Mixed internal and external links.
3. Standard Markdown headings.
4. Ordered and unordered lists.
5. A posts section that demonstrates list and single-page layouts.
